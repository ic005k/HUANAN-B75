
华南金牌B75黑苹果配置（目前以OC引导为主，Clover的引导不再维护）
---

* 主板：华南金牌B75（2019.10.29的BIOS版本）

* CPU：E3 1220 V2

* 内存：金士顿DDR3 1333 2G x 2

* 显卡：迪兰恒进HD7770 1G（引导参数添加radpg=15，配合WhateverGreen.kext免驱）

* 硬盘：威刚SP580 120G SSD

* MacOS：10.15.7（S1睡眠正常，CPU变频正常）
